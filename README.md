# 山姆大叔

歡迎來到我的 `Git Page`，我是山姆大叔。山姆大叔是一個非常迷戀技術與協作的人，因此常常思考要如何才可以讓技術團隊可以更快速而有效的交付。

過去山姆大叔的經驗大多都是在`Azure DevOps Service`上設計協作與交付流程，而且大多數的私有或公有專案都在`Azure DevOps Service`上。但 `Git hub` 實在是太香了，甚至可以用來做公開的網頁。因此山姆大叔決定要在這裡建立相關知識的分享介紹。
